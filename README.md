# Exploratory Data Analysis (EDA) on Retail Sales Data

##  Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a retail sales dataset
to understand sales trends, customer behavior, and product performance.
The goal is to extract meaningful insights that can help businesses make data-driven decisions.

##  Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Dataset
The dataset contains retail sales information including:
- Transaction dates
- Customer demographics
- Product categories
- Sales amounts

##  Key Analysis Performed
- Data loading and cleaning
- Descriptive statistical analysis
- Time series analysis of sales trends
- Customer demographic analysis
- Product category performance analysis
- Data visualization using charts and heatmaps

##  Key Insights
- Sales are concentrated in specific product categories
- Monthly sales trends indicate seasonal patterns
- Customer demographics influence purchasing behavior

##  Business Recommendations
- Focus inventory planning on high-performing product categories
- Use customer insights for targeted marketing
- Plan promotions during low-sales periods

##  How to Run the Project
1. Clone the repository
2. Open the Jupyter Notebook
3. Run all cells sequentially

##  Project Files
- `Task1(OIB-SIP)-EDA.ipynb` – Main analysis notebook  
- `retail_sales_dataset.csv` – Dataset used for analysis
